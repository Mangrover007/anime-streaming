# AniTube - Anime Streaming Website

## Overview
A full-stack anime streaming platform to watch the latest anime shows built using PERN stack.

## Key Features
- User Authentication
- Video Streaming
- Admin and User roles
- Comments

## Tech Stack
- PostgreSQL
- Express
- React
- NodeJS
